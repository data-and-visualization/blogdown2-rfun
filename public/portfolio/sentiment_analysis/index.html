Sentiment Analysis

A brief introduction to text mining and sentiment analysis with visualization. Analyze public domain novels by Jane Austen, wrangle text-data into submission, tokenize corpora, visualize sentiment and generate word clouds.

What's this all about?

Rfun is a campus/community-oriented data science training-series focusing on learning the R programming language and the Tidyverse ecosystem. We are hosted by the Center for Data & Visualization Sciences http://library.duke.edu/data
