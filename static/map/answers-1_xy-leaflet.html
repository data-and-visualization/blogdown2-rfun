Exercise 1 Answers: Georeferencing

library(tidyverse)
library(leaflet)

Load Data

xy_locations <- read_csv("data/mockaroo_latlon.csv")

Plot XY on the Map

xy_locations %>% 
  leaflet(width = "100%") %>% 
  addTiles() %>% 
  addMarkers(lat = ~Latitude, 
             lng = ~Longitude, 
             popup = xy_locations$Company)
