Shapefiles as sf

Using the tigris package get Census Tiger shapefiles for census geographies. Tigris will return the shapefile in the sf, or simple features, format.

us_geo <- tigris::states(class = "sf")
# us_geo_spdf <- states()  # spdf uses @data slots -- old school

Data Structure

The advantage of importing data as simple features (i.e. sf) over sp is that sp uses the old school @data slots method. If you’re used to this it can be a highly functional data structure. But if you’re inclined toward the tidyverse method, you’ll see your data in a more familiar and easy-to-view format, i.e. a data frame rather than a slot. Below are two methods of viewing the downloaded shapefiles.
